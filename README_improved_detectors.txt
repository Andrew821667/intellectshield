Улучшенные детекторы для проекта intellectshield

Обзор

Этот репозиторий содержит улучшенные версии детекторов аномалий и SQL-инъекций для проекта intellectshield. Были внесены значительные улучшения, повышающие эффективность обнаружения аномалий и вредоносных SQL-запросов.

Улучшенные детекторы

1. Улучшенный детектор SQL-инъекций (ImprovedSQLInjectionDetector)

Основные улучшения:
- Расширенный набор паттернов для обнаружения различных типов SQL-инъекций
- Категоризация паттернов с различными весами для более точного определения уровня риска
- Статистический анализ характеристик запросов для обнаружения аномалий
- Адаптивный порог обнаружения, который автоматически настраивается на основе распределения аномальных оценок
- Контекстный анализ, учитывающий поведение IP-адресов
- Подробные объяснения и рекомендации по каждой обнаруженной инъекции

2. Улучшенный ансамблевый детектор (ImprovedEnsembleDetector)

Основные улучшения:
- Различные стратегии ансамблирования:
  - Взвешенное голосование (weighted_voting)
  - Стэкинг (stacking) - использование мета-модели для обучения на результатах базовых детекторов
  - Слияние на основе рангов (rank_fusion) - объединение результатов через ранжирование
- Автоматический выбор оптимальной стратегии на основе метрик производительности
- Гибкая система весов для различных детекторов
- Расширенные метрики оценки эффективности
